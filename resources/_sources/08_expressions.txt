#############################
Routines and Expressions
#############################

Logical expression blocks
=============================

::

   while [ logical expr ]
   do
     code statements
   done

::
 
   until [ logical expr ]
   do
     code statements
   done

::
 
   case "$variable" in
     value1)
       code statements ;;
     value2)
       code statements ;;
   esac
